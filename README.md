# Customer Churn Analysis

## Project Overview
End-to-end customer churn analysis using SQL, Python, and Power BI.
Built a Random Forest model to predict churn for newly joined customers
and created an interactive Power BI dashboard for business insights.

## Tools & Technologies
- SQL (MySQL)
- Python (Pandas, Scikit-learn)
- Power BI
- Jupyter Notebook

## Project Structure
- data/ → Raw and prediction datasets
- sql/ → SQL queries and views
- notebooks/ → Model building and analysis
- powerbi/ → Power BI dashboard
- dashboard_screenshots/ → Dashboard images
